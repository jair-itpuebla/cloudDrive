¿Cómo podemos ayudarte?

El soporte de Cloud Drive te ofrece asistencia para solucionar tus dudas.

Como usar Cloud Drive

    Para hacer uso de Cloud drive debes:
  • Iniciar sesión colocando tu correo electrónico y tu password.
  • Si aún no cuentas con una cuenta debes crear una.
  • Cuando inicies sesión podrás almacenar el contenido que desees dependiendo el plan que tengas.

Crear cuenta

    Para crear tu cuenta necesitas:
  • Seleccionar el apartado de crear cuenta.
  • Llenar todos los campos que te pide que son: nombre, apellido, fecha de nacimiento, correo electrónico, crear una password, seleccionar tu sexo.
  • Al finalizar el llenado debes darle guardar y se creará tú cuenta.

Olvidé mi password

    Si olvidaste tu password debes:
  • Seleccionar iniciar sesión.
  • Colocar tu correo electrónico.
  • Seleccionar olvidé mi password.
  • Automáticamente llegará a tu correo un enlace para cambiar tu password.
